# Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free and respectful experience for everyone.

## Our Standards

Examples of positive behavior include:

- Respectful communication
- Constructive feedback
- Ethical cybersecurity practices
- Supporting other contributors
- Professional collaboration

Examples of unacceptable behavior include:

- Harassment or discrimination
- Malicious activities
- Offensive comments
- Sharing harmful or illegal content
- Personal attacks

## Enforcement

Project maintainers may remove comments, commits, issues, or contributions that violate this Code of Conduct.

## Scope

This Code of Conduct applies to all project spaces, including GitHub discussions, issues, pull requests, and community interactions.

## Contact

For concerns or reporting issues, contact the project maintainer.

---

This project supports ethical and defensive cybersecurity research only.
